# 🚀 Modern Portfolio - Chinmai J

A sleek, responsive personal portfolio built with **HTML5**, **CSS3 (Glassmorphism)**, and **Bootstrap 5**, served as a web application using **Gradio**.

![Project Status](https://img.shields.io/badge/Status-Active-brightgreen)
![Tech Stack](https://img.shields.io/badge/Stack-HTML%20%7C%20CSS%20%7C%20JS%20%7C%20Python-blue)

## ✨ Features
- **Modern UI:** Uses Inter fonts and a custom purple-to-blue gradient palette.
- **Glassmorphism:** Elegant, semi-transparent design elements for a high-tech feel.
- **Responsive:** Mobile-first design using Bootstrap 5's grid system.
- **Python Integration:** Served via Gradio for easy local hosting and sharing.
- **Accessibility:** Optimized with ARIA labels and SEO-friendly tags.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5, FontAwesome Icons.
- **Backend/Hosting:** Python, Gradio (for serving static files).
- **Fonts:** Inter (via Google Fonts).

## 🚀 Getting Started

### Prerequisites

Make sure you have Python installed. You will also need the Gradio library:

```bash
pip install gradio

```

### Installation & Run

1. **Clone the repository:**
```bash
git clone [https://github.com/chinmaicj1610/portfolio.git](https://github.com/chinmaicj1610/portfolio.git)

```


2. **Navigate to the folder:**
```bash
cd portfolio

```


3. **Launch the app:**
```bash
python app.py

```


4. **View in Browser:**
Open the local URL provided by Gradio (usually `http://127.0.0.1:7860`).

## 🛡️ Security & Best Practices

This project follows modern web standards:

* All external links use `rel="noopener"` for security.
* Icons include `aria-label` for screen reader accessibility.
* CSS uses standard properties with vendor prefixes for cross-browser compatibility.
